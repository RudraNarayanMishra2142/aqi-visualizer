CA1 Assignment Documentation

1. Introduction & Problem Statement

Rapid urbanization and industrial expansion have positioned air pollution as one of the most severe public health crises of the 21st century. The core problem is an information gap: citizens need to know what scientific numbers mean for their daily lives. This project bridges that gap by engineering a real-time AQI Visualization Command Center.

2. Review of Existing Literature

Historically, access was limited to static daily reports. This project builds upon modern UI/UX paradigms, utilizing real-time API fetching to eliminate latency and implementing dynamic color theory to instantly communicate threat levels without requiring the user to interpret the underlying chemistry.

3. Description of the Product

  • Live Telemetry Hub: Color-coded central dashboard.
  • Comprehensive Diagnostics: A 10-point grid analyzing primary pollutants.
  • Temporal Roadmap: An interactive historical trajectory timeline.
  • Lifestyle Matrix: Algorithmic behavioral advice.
