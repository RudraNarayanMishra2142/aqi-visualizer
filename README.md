## 📖 About the Project
Rapid urbanization has positioned air pollution as one of the most severe public health crises of the 21st century. The core problem is an information gap: citizens need to know not just raw scientific numbers, but what those numbers mean for their daily lives. 

This project bridges that gap by engineering a highly accessible, real-time AQI Visualization Command Center that translates complex atmospheric data into clear visual warnings and actionable lifestyle advice.

## ✨ Key Features
* **Live Telemetry Hub:** Uses GPS reverse-geocoding to fetch your exact local Air Quality Index, dynamically shifting the UI theme from Safe (Green) to Hazardous (Red).
* **Deep Diagnostics:** A comprehensive 10-pollutant telemetry grid tracking PM2.5, PM10, NO2, O3, SO2, CO, and simulated historical constraints.
* **Forecast Hub:** A 72-hour predictive line chart paired with an algorithmic Lifestyle Matrix (advising on outdoor exercise, ventilation, and mask usage).
* **Temporal Roadmap:** An interactive, branching timeline visualizing India's atmospheric history from 2011 to projected sustainable or dystopian outcomes in 2041.
* **Dynamic Environment Engine:** Features a cinematic boot-loader and automatically syncs the UI to Day/Night mode based on the user's local clock.

## 🛠️ Technology Stack
* **Frontend:** HTML5, CSS3, Vanilla JavaScript (ES6+)
* **Data Visualization:** Chart.js
* **Animations:** Particles.js
* **APIs Used:** Open-Meteo (Air Quality & Weather), BigDataCloud (Reverse Geocoding), DiceBear (Avatars)
